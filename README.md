# Maze Navigation AI Agent

This project implements an intelligent agent that learns how to navigate a maze using reinforcement learning.

## Overview

The agent starts without prior knowledge and learns to find the optimal path from start to goal through interaction with the environment.

It improves over time using rewards and penalties.

## Features

- Q-learning based decision making
- Exploration vs exploitation strategy
- Goal-based, utility-based, and learning agent
- Stores knowledge in a Q-table
- Uses BFS algorithm as a reference for optimal path
- Visualized using HTML, CSS, and Canvas

## Technologies Used

- Python
- NumPy
- Flask
- HTML / CSS / JavaScript (Canvas)
- Kaggle dataset (maze data)

## How It Works

- The agent explores the maze
- Receives rewards and penalties
- Updates Q-values based on experience
- Gradually learns the best path
